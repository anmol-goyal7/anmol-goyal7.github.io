## ğŸš€ Running Locally

To test this website on your own machine:

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/anmol-goyal7/anmol-goyal7.github.io.git](https://github.com/anmol-goyal7/anmol-goyal7.github.io.git)
    cd anmol-goyal7.github.io
    ```

2.  **Install dependencies:**
    (Make sure you have Ruby and Bundler installed)
    ```bash
    bundle install
    ```

3.  **Run the server:**
    ```bash
    bundle exec jekyll serve
    ```

4.  Open your browser and go to `http://localhost:4000`.
