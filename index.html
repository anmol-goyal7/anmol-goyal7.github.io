01

matrix openmp benchmark

benchmarks serial vs parallel matrix multiplication to demonstrate the raw power of multi-core CPUs. achieved a 4.72× speedup over sequential execution using OpenMP thread-level parallelism.

C OpenMP parallel computing benchmarking

02

ray tracer in C++

a physically-based ray tracer built from scratch — no libraries, just math and pixels. implements sphere intersections, diffuse / specular shading, and anti-aliasing via supersampling.

C++ ray tracing graphics linear algebra
github ↗
03

software rasterizer

a 3D rasterizer built from first principles following tinyrenderer — renders .obj files to a TGA framebuffer. implements barycentric coordinates, z-buffering, Gouraud shading, and texture mapping.
